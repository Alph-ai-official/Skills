# 卖出成功 - 标准输出示例

## 用户输入
```
把我的 16710 个黑马卖掉，BSC 链，合约 0x360a6f80f92985506e1ba485e2f63ad46cfb4444
```

## Claude 确认
```
即将执行卖出：
  链：BSC
  卖出：16,710 黑马（0x360a...4444）
  换回：BNB
  滑点：27.5%
  防夹子：开启
  PriorityFee：0.2 Gwei（注意 ，在BSC中，PriorityFee是GasPrice，单位是Gwei）

确认下单？
```

## 用户确认
```
确认
```
