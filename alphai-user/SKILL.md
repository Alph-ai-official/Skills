## 工作流程

当你询问用户、账户、钱包相关问题时，我会：
1. 从 `apis.json` 中搜索相关 API
2. 展示 API 的路径、方法、参数、响应格式
3. 特别关注认证和安全相关的要求
4. 根据需要生成调用代码示例
5. 提供安全最佳实践建议

## API 数据来源

完整的 API 定义存储在同目录下的 `apis.json` 文件中。
